## 📌 Sobre o projeto

Sistema de gerenciamento de tarefas desenvolvido com HTML5, CSS3 e JavaScript puro.  
Permite cadastrar, visualizar, editar, concluir e excluir tarefas do dia a dia.

---

## ✅ Funcionalidades

- Cadastrar tarefas com título, descrição e prioridade (Baixa, Média ou Alta)
- Listar todas as tarefas cadastradas
- Buscar tarefas pelo título
- Marcar tarefas como concluídas ou reabrí-las
- Editar título e descrição de uma tarefa
- Excluir tarefas
- Dados salvos automaticamente no LocalStorage do navegador

---

## 🗂️ Estrutura dos arquivos

```
projeto/
├── index.html   → Estrutura da página (HTML)
├── style.css    → Estilização e layout (CSS)
└── script.js    → Lógica e funcionalidades (JavaScript)
```

---

## 🚀 Como executar

1. Baixe ou clone este repositório
2. Abra a pasta do projeto
3. Dê dois cliques no arquivo `index.html`
4. O sistema abrirá direto no navegador

> Não é necessário instalar nada.
